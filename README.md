# LLB Mobile

Flutter MVP мобильного приложения для `llb.su`.

## Что изучено на llb.su

Основные разделы сайта:

- `Турниры`: текущие, предстоящие, завершенные.
- `Рейтинги`.
- `Каталоги`: люди, команды, клубы, федерации, школы и тренеры.
- Карточка турнира ведет к участникам, сетке, фото, видео и связанным блогам.

Главная сущность приложения:

- Турнир: название, город, клуб, дисциплина, уровень, дата, статус, участники, сетка, матчи.
- Игрок: имя, город, клуб, дисциплина, рейтинг.

## Что уже сделано

- Нижняя навигация: турниры, игроки, рейтинги, API.
- Фильтр турниров по статусу: текущие, скоро, итоги.
- Поиск по турнирам и игрокам.
- Детальная карточка турнира с участниками и матчами.
- Mock-репозиторий с данными в структуре LLB.
- `ChallongeApiClient` для проверки API key и загрузки турнира, участников и матчей.

## API

В текущем MVP данные на экранах идут из `MockLeagueRepository`.

Для Challonge добавлен клиент в `lib/main.dart`:

- `GET /tournaments/{tournament}.json`
- `GET /tournaments/{tournament}/participants.json`
- `GET /tournaments/{tournament}/matches.json`

Экран `API` позволяет вручную вставить Challonge API key и slug/id турнира, чтобы проверить подключение.

## Импорт данных LLB

Добавлен локальный скрейпер:

```bash
export LLB_USERNAME='...'
export LLB_PASSWORD='...'
python3 tools/llb_scraper/llb_scraper.py --login sample-all
```

Он складывает игроков, карточки игроков, турниры, стадии, участников и матчи в
`data/llb.sqlite3`. Подробные команды есть в `tools/llb_scraper/README.md`.
